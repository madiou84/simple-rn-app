# Simple App RN

## Installation

Clone the repo locally:

```sh
git clone https://github.com/madiou84/simple-rn-app.git simple-rn-app
cd simple-rn-app
```

Install JavaScript dependencies:

```sh
yarn install
```
OR
```sh
npm install
```

# Run on mobile

For android

```sh
mpx react-native run-android
```
